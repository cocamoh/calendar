In this example you see how does the SWL sticky-list-headers div extension works. All you need to do is to import the library, which can be found on GitHub and define your parent div to be sticky-list-headers. Lastly you need to define a selector for your headers which could be for example a class name. The selector should be in the form of the JavaScript function element.querySelectorAll.

...
<script src="../src/sticky-list-headers.js"></script>
...
<div is="swl-sticky-list-headers" selector=".header" ...>
...
</div>

It could be that it is not working when you try this example in your page. That is because this library uses HTML5 custom elements and shadow root. Hence try to load the polyfills from Polymer.

...
<script src="http://cdnjs.cloudflare.com/ajax/libs/polymer/0.3.4/platform.js"></script>
<script src="http://cdnjs.cloudflare.com/ajax/libs/polymer/0.3.4/polymer.js"></script>
...

Lastly do not forget to apply your header styles to the shadow root (::shadow).

...
.header,
::shadow .header {
	position: relative;
	z-index: 1;
	padding-bottom: 10px;
	background-color: #C7AD66;
	border: 1px solid #573F00;
	-moz-box-shadow: 0 0 8px #573F00;
	-webkit-box-shadow: 0 0 8px #573F00;
	box-shadow: 0 0 8px #573F00;
}
...
